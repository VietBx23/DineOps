# DineOps Frontend

React + Vite frontend skeleton for DineOps.

## Setup

- `cd frontend`
- `npm install`
- `npm run dev`

## Pages

- `src/App.jsx`
- `src/pages/HomePage.jsx`
- `src/components/Navbar.jsx`
- `src/api/index.js`

## Notes

- API calls can be added in `src/api/`
- Update `src/App.jsx` to connect to backend
